# EcoQuest - Static Frontend for Environmental Learning

A static HTML/CSS/JavaScript web application for reviewing environmental science lessons and quizzes.

## Features

- Static pages with hardcoded data
- Lessons on environmental topics
- Quizzes with feedback
- Simple navigation

## Tech Stack

- Frontend: HTML, CSS, JavaScript

## Usage

Open `public/index.html` in your browser to start.

## Project Structure

- public/: Static files (HTML, CSS, JS)

## UI Design Idea

- Color scheme: Green (#4caf50) and white for eco-friendly feel
- Layout: Clean, simple design with navigation

- Add more lessons and quizzes
- Improve styling with animations
